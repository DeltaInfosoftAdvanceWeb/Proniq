## üé® FORMATTING GUIDELINES

### Headings

```
H1: Primary Keyword (Only ONE per page)
  Font: 48-60px
  Weight: Bold
  Example: "Construction ERP Software for Contractors"

H2: Section Headings (3-5 per page)
  Font: 32-40px
  Weight: Bold
  Example: "How to Reduce Billing Cycles"

H3: Subsection Headings
  Font: 24-28px
  Weight: Semibold
  Example: "Automated Measurement Entry"
```

### Paragraphs

- **Length:** 2-4 sentences max
- **Line height:** 1.6-1.8
- **Font size:** 16-18px
- **Color:** Dark gray (#334155), not pure black

### Lists

- Use bullet points for features/benefits
- Use numbered lists for steps/processes
- Keep items parallel in structure
- Start each item with action verb when possible
